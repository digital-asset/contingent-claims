About this Data Dictionary

This document contains information about CHESS+ system internals, specifically it lists all CHESS+ contract templates, related data elements, their attributes, meanings, and permitted values. The Data Dictionary consists of these sections:

  • Access API: Lists available API Fact contracts, which are created on the ledger and are accessible via the Ledger API by authorised participants.
  • Ingress Contracts: Lists available Master Ingress contracts and choices available within each.
  • Egress Contracts: Lists available Master Egress contracts and choices available within each.
  • Business Models: Lists all CHESS+ business workflow templates, available choices, fields, and data types used within a workflow.
  • Language Types: Lists details of all the CHESS+ data types called by templates listed in the Business Models section.
  • Messages: Lists DAML templates for all ISO20022 XML messages supported by CHESS+.
