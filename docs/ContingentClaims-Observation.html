eval

: (Number x, Divisible x, Action m) => (a -> t -> m x) -> Observation t x a -> t -> m x

Reify the Observable into an observation function. The function is only total when the first argument is too (typically it will fail on t > today).

mapParams

: (t -> i) -> (a -> a') -> (x -> x') -> Observation i x a -> Observation t x' a'

The functor map operation and also map any parameters to keys. For example, could map the param "spot" to an ISIN code "GB123456789". Also contra-maps time parameter, i.e. from relative time values to absolute ones.
